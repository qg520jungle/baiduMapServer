用于批量处理地址，以在百度地图上标示出来

使用 node.js 生成服务器 作为后台处理页面传回的数据
其中涉及 文件上传，文件下载，路由跳转，读取文件，生成文件(正在做注册，登陆功能)

引用 bootstrap fileinput 插件，上传文件

使用 baidu 地图的 API 实现页面多地址同时显示 (暂未做城市切换功能，默认为杭州市地址)

登陆注册设计了一个好玩儿的交互，用 animate 实现

本应用已发布到百度云服务器 可通过 https://qg520jungle.duapp.com 在线查看演示效果
